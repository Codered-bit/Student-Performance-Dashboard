Student Performance Analytics Dashboard

## Overview

This project simulates institutional student data to analyse academic performance, attendance patterns, and risk segmentation. It demonstrates end-to-end data analytics workflow from data generation to dashboard deployment.

## Features

- Synthetic institutional dataset generation

- Data cleaning and preprocessing pipeline

- KPI analysis by department

- Student risk segmentation model

- Interactive Streamlit dashboard

## Tech Stack

Python, Pandas, NumPy, Seaborn, Matplotlib, Streamlit

## Key Insights

- Attendance shows moderate correlation with final grades

- Students with <50% grades classified as high risk

- Department-level performance variation identified

pip install -r requirements.txt
